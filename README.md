This repository provides a modular gamepad interface for Duatic products, enabling intuitive control of robots like the DynaArm. It supports various controller types and features a dynamic focus mechanism for controlling different robotic components (e.g., arms, hip, platform).

## 📚 Documentation

For detailed instructions on how to use the gamepad with your robot, please refer to the official documentation:

👉 [**DynaArm Operator Manual — Control with a Gamepad**](https://docs.duatic.com/dynaarm/operators/robot_usage/robot_usage.html#control-with-a-gamepad)

## 🚀 Quick Start

To run the gamepad interface with the default configuration:

```bash
ros2 run duatic_gamepad_interface gamepad_interface
```
